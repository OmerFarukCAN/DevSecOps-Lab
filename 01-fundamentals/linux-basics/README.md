# Linux Temelleri

**Süre:** 5-7 gün  
**Seviye:** Başlangıç  
**Ön Koşul:** Yok

## Öğrenme Hedefleri

Bu modülü tamamladığınızda:
- ✅ Linux komut satırında rahatça çalışabileceksiniz
- ✅ Dosya sistemi yapısını anlayacaksınız
- ✅ Temel sistem yönetimi yapabileceksiniz
- ✅ Process'leri yönetebileceksiniz
- ✅ Basit bash script'leri yazabileceksiniz
