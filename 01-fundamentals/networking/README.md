# Networking Temelleri

**Süre:** 5-7 gün  
**Seviye:** Başlangıç-Orta  
**Ön Koşul:** Linux Basics

## Öğrenme Hedefleri

DevSecOps engineer olarak network'ü anlamak ZORUNLU! Bu modülde:
- ✅ OSI ve TCP/IP modellerini anlayacaksınız
- ✅ IP adresleme ve subnetting yapabileceksiniz
- ✅ Network troubleshooting yapabileceksiniz
- ✅ DNS ve HTTP protokollerini anlayacaksınız
- ✅ Firewall ve security group'ları yönetebileceksiniz
